# National EV Infrastructure Readiness Platform

## Project Overview

This project aims to identify priority Indian cities for future Electric Vehicle (EV) charging infrastructure investments using data analytics.

The analysis integrates multiple factors such as population, EV registrations, charging infrastructure, commercial activity, transportation accessibility, and urban development to provide data-driven recommendations.

---

## Problem Statement

India is witnessing rapid growth in electric vehicle adoption, but charging infrastructure expansion has not kept pace across all cities.

Government agencies and charging network operators require a data-driven approach to identify infrastructure gaps and prioritize cities for future investments.

This project develops an analytics platform to evaluate EV infrastructure readiness across Indian cities.

---

## Project Objectives

- Analyze EV adoption across Indian cities.
- Identify infrastructure gaps.
- Compare cities using key performance indicators.
- Generate business insights for future charging station investments.
- Build interactive Power BI dashboards.

---

## Dataset

The project uses a synthetic dataset inspired by:

- VAHAN Dashboard
- Census of India
- OpenStreetMap
- Bureau of Energy Efficiency (BEE)
- Ministry of Road Transport & Highways (MoRTH)
- State EV Policy reports
